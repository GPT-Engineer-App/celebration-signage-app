# celebration-signage-app

Digital signage app for celebrating displaying birthdays, work anniversaries, and holiday announcements. Make one UI that displays on a smart TV to show the signage and one for administration where the data can be edited. Include mock data to start with.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/celebration-signage-app.git
cd celebration-signage-app
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
